Variables & Types

A variable is something that holds a value that may change. In simple terms, is just an empty box where you can put stuff in. You can use variables to store all kinds of stuff, but in the following exercises we are just going to be storing strings and numbers in variables.

In JavasScript we use the keyword var to define a variable.

Variables have different types:

  • String var myString = "Im a String";
  • Number var myNumber = 6;
  • Boolean var myBoolean = true;
  • List var myFruitsList = ['apple', 'orange', 'grape'];
  • Object var myStudentObject = {name: 'James', age: 12};

- In JavaScript, the Number type can be both a decimal number and an integer.

- String text must be inside double quotes " " or single quotes ' '.

- Boolean variables can only be equal to either true or false.

Math Operators

In JavaScript we can execute regular math operations. Such as:

Addition operator

The + (addition) operator is used both addition and concatenation(combining texts together) of strings.

For Example, Adding numbers:

var a = 1;
var b = 2;
var c = a + b; // c is now equal to 3

The addition operator is used for concatenating strings to strings, strings to numbers, and numbers to strings:

var name = "Michael";
console.log("Hello " + name + "!");
console.log("My grade is " + 95);
console.log(95 + " is my grade");
Other Arithmetic Operators

To subtract use the minus (-), multiply use asterisk (*), and divide use slash (/) signs.

console.log(3 - 5); // outputs -2
console.log(3 * 5); // outputs 15
console.log(3 / 5); // outputs 0.6

Arrays

Arrays are used to store multiple values in a single variable. To define an Array in JavaScript we use the brackets notations [ ] .

For Example:

//Here we are declaring an Array
var myFavoriteColors = ["purple", "green", "red"]; //this array contains string values
var myFavoriteNumbers = [12,6,2,7,5]; //this array contains number values

We can use array literal [ ] to access an specific value in our array. Arrays indexes start from 0, for instance:

//Let's assign the first value of the array myFavoriteNumbers to the variable firstValue
var firstValue = myFavoriteNumbers[0]; // firstValue now contains the number 12;

Conditions

Conditions allow us to check if a statement is valid, and execute different code according to the result.

For Example: if we want know whether a number is hihger than other. We can write a condition to check.

Logical Operators

Two or more expressions can be evaluated together using logical operators to check if two expressions evaluate to true together, or at least one of them. To check if two expressions both evaluate to true, use the AND operator &&. To check if at least one of the expressions evaluate to true, use the OR operator ||.

For Example:

var foo = 1;
var bar = 2;
var moo = 3;
if (foo < bar && moo > bar)
{
console.log("foo is smaller than bar AND moo is larger than bar.");
}
if (foo < bar || moo > bar)
{
console.log("foo is smaller than bar OR moo is larger than bar.");
}
